  <action>
    Create directory `frontend/src/hooks/aiDiscover/` and file `useAiDiscover.ts`.

    This hook wraps `postAiDiscover` in a `useMutation` from React Query v5. It must expose the mutation state so consumers can:
    - Call `mutate(query)` to submit a query string
    - Read `data` for the response
    - Read `isPending` for loading state
    - Read `error` to check error status codes

    Implementation:

    ```typescript
    import { useMutation } from "@tanstack/react-query";
    import { postAiDiscover } from "../../api/aiDiscover.api";
    import type { AiDiscoverResponse } from "../../types/aiDiscover";
    import type { AxiosError } from "axios";

    export function useAiDiscover() {
      return useMutation<AiDiscoverResponse, AxiosError, string>({
        mutationFn: (query: string) => postAiDiscover({ query }),
      });
    }
    ```

    The generic params are `<TData, TError, TVariables>`:
    - TData = `AiDiscoverResponse` (success payload)
    - TError = `AxiosError` (so consumers can read `error.response?.status` for 503/429/400)
    - TVariables = `string` (the query text)

    No `onSuccess`/`onError` callbacks — the page component handles UI state transitions.
