Constructor dependencies (all scoped or resolved per-scope):
- `AppDbContext _db`
- `ITmdbService _tmdbService`
- `AzureOpenAIClient _azureClient`
- `AzureOpenAIOptions _options`
- `ILogger<EmbeddingSeedService> _logger`

Constants:
- `const int MovieTarget = 10_000;`
- `const int TvTarget = 5_000;`
- `const int TotalTarget = 15_000;`
- `const int MaxMoviePages = 500;`  (D-07: 20/page * 500 = 10,000)
- `const int MaxTvPages = 250;`  (D-08: 20/page * 250 = 5,000)
- `const int BatchSize = 50;`  (D-12)
- `const int ThrottleDelayMs = 250;`  (discretion: TMDB throttle)
- `const int TitlesPerPage = 20;`

**`RunAsync` method:**
1. Count existing rows: `var movieCount = await _db.MovieEmbeddings.CountAsync(e => e.MediaType == "movie", ct);` and same for TV (D-20 checkpoint)
2. Check if complete: `if (movieCount + tvCount >= TotalTarget)` â†’ log "Seed complete, {movieCount} movies + {tvCount} TV shows" at INFO and return (D-19)
3. Get embedding client: `var embeddingClient = _azureClient.GetEmbeddingClient(_options.EmbeddingDeployment);`
4. If `movieCount < MovieTarget` â†’ call `SeedMediaTypeAsync("movie", movieCount, MaxMoviePages, embeddingClient, ct)` (D-09: movies first)
5. If `tvCount < TvTarget` â†’ call `SeedMediaTypeAsync("tv", tvCount, MaxTvPages, embeddingClient, ct)`
6. Log final counts

**`SeedMediaTypeAsync` method** (private):
Parameters: `string mediaType, int existingCount, int maxPages, EmbeddingClient embeddingClient, CancellationToken ct`

1. Calculate resume page: `int startPage = (existingCount / TitlesPerPage) + 1;` (D-20)
2. Log: "Starting {mediaType} seed from page {startPage} ({existingCount} existing)"
3. Initialize: `var batch = new List<(int TmdbId, string ContentText, MovieEmbedding Entity)>(BatchSize);`
4. Loop `for (int page = startPage; page <= maxPages && !ct.IsCancellationRequested; page++)`:
   a. Fetch page: `var pageJson = mediaType == "movie" ? await _tmdbService.GetPopularMoviesPageAsync(page) : await _tmdbService.GetPopularTvPageAsync(page);`
   b. Parse TMDB IDs from page JSON: deserialize to extract `results[].id` array using `JsonDocument.Parse(pageJson)` then `root.GetProperty("results").EnumerateArray().Select(el => el.GetProperty("id").GetInt32())`
   c. For each ID, if `ct.IsCancellationRequested` break:
      - Try/catch per title (Pitfall 4):
        - Fetch detail: typed async method (`GetMovieDetailsForSeedTypedAsync` or `GetTvDetailsForSeedTypedAsync`)
        - Skip check (D-11): if `detail is null || (string.IsNullOrEmpty(detail.Overview) && (detail.Genres == null || detail.Genres.Count == 0))` â†’ log WARNING "Skipping {mediaType} {id}: missing overview and genres" and continue
        - Build content text: `EmbeddingContentBuilder.BuildMovieText(detail)` or `BuildTvText(detail)`
        - Build MovieEmbedding entity (populate Title, Overview, Genres as comma-separated, Keywords as comma-separated, CastCrew as comma-separated, ReleaseYear parsed from date, VoteAverage, ContentText, MediaType)
        - Add to batch
      - Catch Exception: log WARNING per title, continue
      - `await Task.Delay(ThrottleDelayMs, ct);` (TMDB throttle)
   d. If `batch.Count >= BatchSize` â†’ call `EmbedAndUpsertBatchAsync(batch, embeddingClient, ct)` then `batch.Clear()`
   e. Log progress every 10 pages: "Seed {mediaType}: page {page}/{maxPages}, ~{existingCount + processedCount} total"
5. Flush remaining batch if non-empty
6. Log completion: "Seed {mediaType} complete"

**`EmbedAndUpsertBatchAsync` method** (private):
Parameters: `List<(int TmdbId, string ContentText, MovieEmbedding Entity)> batch, EmbeddingClient embeddingClient, CancellationToken ct`

1. Extract content texts: `var inputs = batch.Select(b => b.ContentText).ToList();`
2. Try:
   a. Call Azure OpenAI: `ClientResult<OpenAIEmbeddingCollection> result = await embeddingClient.GenerateEmbeddingsAsync(inputs, cancellationToken: ct);`
   b. Validate count (Pitfall 3): `if (result.Value.Count != batch.Count)` â†’ log WARNING "Embedding count mismatch: expected {batch.Count}, got {result.Value.Count}" and return
   c. For each index `i`:
      - `ReadOnlyMemory<float> floats = result.Value[i].ToFloats();`
      - `batch[i].Entity.Embedding = new Pgvector.Vector(floats.ToArray());`
      - Upsert: check `var existing = await _db.MovieEmbeddings.FirstOrDefaultAsync(e => e.TmdbId == batch[i].TmdbId && e.MediaType == batch[i].Entity.MediaType, ct);`
      - If existing: update ContentText, Embedding, UpdatedAt, and metadata fields
      - If null: `_db.MovieEmbeddings.Add(batch[i].Entity);`
   d. `await _db.SaveChangesAsync(ct);`
   e. Log INFO: "Embedded batch of {batch.Count} {batch[0].Entity.MediaType} titles"
3. Catch `Azure.RequestFailedException ex when (ex.Status == 429)`: log ERROR "429 rate limit exhausted after SDK retries for batch" (D-13, D-14)
4. Catch `Azure.RequestFailedException ex`: log ERROR "Azure OpenAI error {ex.Status} for batch"
5. Catch `OperationCanceledException`: throw (let cancellation propagate)
6. Catch `Exception ex`: log ERROR "Unexpected error embedding batch"

Model directly on `TmdbRefreshBackgroundService`. Same scaffold:

```csharp
using backend.Services;

namespace backend.BackgroundJobs;

public class EmbeddingSeedBackgroundService : BackgroundService
{
    private readonly IServiceScopeFactory _scopeFactory;
    private readonly ILogger<EmbeddingSeedBackgroundService> _logger;
    private const int StartupDelayMinutes = 5; // D-18: 5-minute startup delay

    public EmbeddingSeedBackgroundService(
        IServiceScopeFactory scopeFactory,
        ILogger<EmbeddingSeedBackgroundService> logger)
    {
        _scopeFactory = scopeFactory;
        _logger = logger;
    }

    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        _logger.LogInformation("Embedding seed service starting, waiting {Delay} minutes before first run",
            StartupDelayMinutes);
        await Task.Delay(TimeSpan.FromMinutes(StartupDelayMinutes), stoppingToken);

        while (!stoppingToken.IsCancellationRequested)
        {
            try
            {
                _logger.LogInformation("Embedding seed cycle starting");
                await RunSeedCycleAsync(stoppingToken);
                _logger.LogInformation("Embedding seed cycle complete");
            }
            catch (OperationCanceledException) when (stoppingToken.IsCancellationRequested)
            {
                break;
            }
            catch (Exception ex)
            {
                _logger.LogError(ex, "Error during embedding seed cycle");
            }

            // D-21: weekly refresh
            await Task.Delay(TimeSpan.FromDays(7), stoppingToken);
        }
    }

    private async Task RunSeedCycleAsync(CancellationToken ct)
    {
        using var scope = _scopeFactory.CreateScope();
        var seedService = scope.ServiceProvider.GetRequiredService<IEmbeddingSeedService>();
        await seedService.RunAsync(ct);
    }
}
```
